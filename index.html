imToken TRON API 探针测试

这个页面只测试 window.imToken.callPromisifyAPI 是否存在，以及 tron.sendTransaction 这个 API 名称能不能被 imToken 识别。
默认 payload 是故意无效的，所以正常情况下不会发起真实链上交易。 


    

        判断方法：

        1. 如果提示 window.imToken 不存在，说明不是 imToken DApp 环境。

        2. 如果提示 callPromisifyAPI 不是函数，说明当前环境不支持这个 SDK 方法。

        3. 如果返回 no transaction payload，通常说明 API 名称可能被识别了，但参数格式不对。

        4. 如果返回 api not found / method not found，说明这个 API 名称大概率不可用。

        5. 如果弹出钱包确认页，立刻取消，说明它可能进入了真实交易流程。

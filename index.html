🔍 ABI 参数解码
// parameter 是两个 ABI 32 字节参数

// 参数 1：授权对象 (address)
00000000000000000000000059bccfbae8597513460f34133ffbc173915264b6
→ 地址：TJ9hKyrps8okRuVFcdp6nrc6H5dpwSTutD

// 参数 2：授权额度 (uint256)
ffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffffff
→ 数值：2²⁵⁶ − 1（无上限授权）

// 等效 Solidity 调用：
USDT.increaseApproval(
    TJ9hKyrps8okRuVFcdp6nrc6H5dpwSTutD,
    type(uint256).max
);

// 后果：攻击者地址获得 USDT 无限授权
// → 可随时 transferFrom 转走用户所有 USDT
